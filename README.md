# рџ“– About

**DairyHub** is a professional Java Desktop Application developed for dairy societies and milk collection centers.

The system manages farmers, daily milk collection, FAT-based milk rate calculation, payment records, reports, receipts, and database backup in one place.

This project is developed using **Java Swing**, **JDBC**, and **MySQL**.
